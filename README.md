# CyberJoss 赛博烧纸 🔥

> 以中国风数字仪式承载思念，让文字、图像与记忆在火光中化为数据之灰。

![CyberJoss Screenshot](https://img.shields.io/badge/CyberJoss-%E8%B5%9B%E5%8D%9A%E7%83%A7%E7%BA%B8-orange?style=for-the-badge)

## ✨ 项目简介

**CyberJoss** 是一个中国风的「赛博烧纸」网页应用。

在数字时代，我们用代码重构传统仪式。用户可以输入祭文、上传文档或图片，系统会根据内容计算"冥力值"，并以逼真的 WebGL 火焰特效、动态音效和仪式感十足的焚化流程，完成一场独特的数字焚化体验。

**核心设计理念：上传并消散** —— 数据在前端分片后发送至后端，后端仅做转发，不存正文、不写日志、不落数据库，最终通过 UDP 投向 RFC 5737 保留地址，作为"数据消散"的技术隐喻。

## 🎬 效果预览

- **逼真的篝火效果**：WebGL 实时渲染，包含焰芯、火星、余烬和烟雾
- **动态火焰响应**：根据内容"冥力值"调整火焰大小和燃烧节奏
- **仪式感流程**：点火 → 燃烧 → 余烬 → 完成，文字在火中逐字浮现、升腾、消散
- **智能速度控制**：小文件放慢保证仪式感，大文件快速完成

## 🚀 核心特性

### 1. 中国风焚化体验
- 🎨 **视觉风格**：深木色、烬红、琥珀金的东方美学配色
- 🔥 **WebGL 火焰**：自定义 GLSL Shader，逼真篝火效果
- ✨ **粒子系统**：飘动的火星和余烬，随火焰摇曳
- 📝 **文字焚化**：祭文在火中逐字浮现、抬升、模糊、消散

### 2. 冥力值系统
| 输入类型 | 计算方式 | 冥力换算 |
|---------|---------|---------|
| 手动输入文本 | 中英文混合字数 | 每 100 字 ≈ 1 冥力 |
| `.md` / `.txt` / `.json` / `.csv` | 解析纯文本 | 每 100 字 ≈ 1 冥力 |
| `.doc` / `.docx` | mammoth 提取文本 | 每 100 字 ≈ 1 冥力 |
| `.pdf` | pdf.js 提取文本 | 每 100 字 ≈ 1 冥力 |
| `.jpg` / `.png` / `.gif` / `.webp` | 按张数计算 | 每张 5 冥力 |

### 3. 火焰等级
| 冥力值 | 等级 | 表现 |
|-------|------|------|
| 0 | 无火 | 余烬微闪 |
| 1-5 | 烛火 | 小火苗，轻微摇曳 |
| 6-20 | 小火 | 火势初成 |
| 21-50 | 中火 | 火焰明显增强 |
| 51-100 | 大火 | 火光满堂，燃烧迅猛 |
| 100+ | 烈焰 | 烈火翻涌，火星更明显 |

### 4. 音效系统
- 🔊 点火音效 — 纸张投入火中的瞬间
- 🔥 持续燃烧循环 — 随火焰大小动态调整
- 🔔 完成钟声 — 余烬消散后的空灵回响
- 🔕 支持静音切换

### 5. 隐私保护
- ❌ 不保存正文内容
- ❌ 不使用数据库
- ❌ 不保留传输日志
- ✅ 仅统计焚烧总次数（不含任何内容）
- 🔒 UDP 转发至 RFC 5737 虚空地址

## 🛠️ 技术栈

### 前端
- **React 18** + **TypeScript**
- **Vite** — 快速构建
- **Tailwind CSS** — 原子化样式
- **Three.js** — WebGL 渲染
- **GLSL Shader** — 自定义火焰效果
- **Howler.js** — 音频控制
- **mammoth** / **pdfjs-dist** — 文档解析

### 后端
- **Node.js** + **Fastify**
- **WebSocket (ws)** — 实时传输
- **dgram (UDP)** — 数据消散

## 🚀 快速开始

### 1. 克隆仓库

```bash
git clone https://github.com/Sprinkler126/cyber-joss.git
cd cyber-joss
```

### 2. 安装依赖

```bash
npm install --prefix client
npm install --prefix server
```

### 3. 启动开发服务器

```bash
# 终端 1：启动后端
npm --prefix server run dev

# 终端 2：启动前端
npm --prefix client run dev -- --host 0.0.0.0
```

### 4. 访问应用

打开浏览器访问 http://localhost:5173

## 🔥 焚化流程

1. **输入内容** — 手写祭文或拖拽文件到火堆
2. **预览火势** — 实时查看对应火焰大小
3. **投入火中** — 文件化为纸卡落入火焰
4. **焚烧过程** — 数据分片发送，文字逐字升腾
5. **余烬消散** — 火势渐熄，完成仪式
6. **数据统计** — 显示本次冥力值与总焚烧次数

## 🎨 设计哲学

**CyberJoss** 的核心不是"上传并保存"，而是"上传并消散"。

在这个数字痕迹无处不在的时代，我们创造了一种**有意识的遗忘仪式**。通过技术实现"数据消散"的隐喻，让数字记忆如纸灰般随风而散。

> "火是净化，也是释放。"
