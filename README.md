## 项目简介

**CyberJoss** 是一个将中国传统祭祀文化数字化的项目。

在现实生活中，我们通过烧纸钱、纸扎来表达对逝者的思念。在数字世界里，**CyberJoss** 让这种情感传递以新的形式延续。

---

## 核心概念

### 什么是「烧纸」？

在中国传统文化中，「烧纸」是一种祭祀仪式：
- 烧纸钱 —— 希望逝者在另一个世界有钱花
- 烧纸扎 —— 房子、车子、手机等，希望逝者生活无忧
- 烧纸衣 —— 换季时为逝者准备衣物

### 什么是「赛博烧纸」？

**CyberJoss** 将这种情感数字化：
- 💳 **数字纸钱** —— 区块链上的虚拟货币转账
- 🏠 **数字纸扎** —— 3D 虚拟物品，NFT 形式永久保存
- 📱 **数字祭品** —— 逝者生前喜爱的数字内容

## 技术栈

- **Frontend**: React + TypeScript + Tailwind CSS
- **Backend**: Node.js + Express
- **Blockchain**: Ethereum / Polygon (NFT 纸钱)
- **Storage**: IPFS (去中心化存储)
- **3D**: Three.js (虚拟纸扎展示)

---

## 名字由来

**CyberJoss** = Cyber (赛博/数字) + Joss (纸钱)

- 「Joss paper」是英文中对纸钱/祭祀纸品的标准称谓
- 「Cyber」代表数字化、网络化
- 组合起来既保留文化根源，又体现技术属性
